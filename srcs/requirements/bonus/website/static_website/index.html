Intro

This website has a single page: index.html.
It is hosted with a docker container running nginx.

How does it work

The nginx server running inside the website container is configured to listen on port 9090. The nginx server running inside the nginx container has a proxy_pass inside the location /static_website/ to pass along requests. Website-nginx also has a location /static_website/ to serve the page.

Website-nginx does not use ssl encryption. It only receives request via the main nginx server. This only accepts ssl requests.
